# TenCor

tenCor is a (hopefully) N-dimensional tensor manipulation and mathematics library written entirely in C, with zero external dependencies.

## Goal
To build a low-level computing core as a foundation for a future machine learning project.

## Directory Structure
* `include/`
  Contains header file `main.h`, defining `struct tensor` data structure and declaring function prototypes.
* `src/`
  Contains source code (`.c`) of the listed functions.
